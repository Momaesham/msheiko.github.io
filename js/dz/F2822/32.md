# Задача

В `html` файле с пустым `body`, использую javascript, создайте структуру из кода ниже

```html
<!doctype html>
<html lang="en">
<head>
     <title>Document</title>
</head>
<body>
<header>
    <nav>
        <ul class="head-menu">
            <li>page1</li>
            <li>page2</li>
            <li>page3</li>
            <li>page4</li>
        </ul>
    </nav>
</header>
<main class="container">
    <article>
        <h1>title</h1>
        <p>content</p>
    </article>
    <aside class="sidebar">
        <ul>
            <li>post1</li>
            <li>post2</li>
            <li>post3</li>
        </ul>
    </aside>
</main>
<footer>
    <p>copyright <b>text</b></p>
</footer>
</body>
</html>
```
